# Tecnologias implementadas en el proyecto:

| Tecnologia |                Descripcion                |
| :--------: | :---------------------------------------: |
|     ==     |                 Es igual                  |
|    ===     |          Es estrictamente igual           |
|     !=     |                Es distinto                |
|    !==     |         Es estrictamente distinto         |
| <,>,<=,>=  | menor,mayor, menor o igual, mayor o igual |
|     &&     |             Operador and (y)              |
|    \|\|    |              Operador or (o)              |
|     !      |             Operador not (no)             |
